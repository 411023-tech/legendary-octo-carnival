# 📚 背單字App - 英文單字學習卡片

一個純前端的英文單字學習應用程式，使用HTML、CSS和JavaScript開發，無需任何框架。

## ✨ 主要功能

### 📖 學習頁面
- **翻轉卡片效果**：點擊卡片可翻頁查看單字資訊
  - 正面：英文單字
  - 背面：中文翻譯、詞性、例句、字根分析
- **前後導航**：使用「上一個」和「下一個」按鈕瀏覽所有單字
- **單字計數**：實時顯示當前位置和總數

### 🛠️ 管理頁面
- **新增單字表單**：輸入單字及其相關資訊
- **🤖 自動填入功能**：
  - 只需輸入英文單字，點擊「自動填入」按鈕
  - 自動調用免費的Dictionary API
  - 自動填入翻譯、詞性、例句、同義詞和詞源信息
- **單字列表**：顯示所有已新增的單字
- **編輯和刪除**：可修改或刪除已新增的單字

### 💾 本地存儲
- 所有數據自動保存到瀏覽器的 localStorage
- 重新整理頁面後資料不會遺失

## 🚀 使用方法

### 安裝和運行

1. **下載或複製文件**
   - 確保三個文件在同一個目錄中：
     - `index.html`
     - `styles.css`
     - `script.js`

2. **在瀏覽器中打開**
   - 直接雙擊 `index.html` 文件，或
   - 用瀏覽器打開：`file:///path/to/index.html`
   - 或在本地服務器上運行（推薦）

3. **使用本地服務器（推薦）**
   ```bash
   # 使用 Python 3
   python -m http.server 8000
   
   # 使用 Python 2
   python -m SimpleHTTPServer 8000
   
   # 使用 Node.js (需要安裝 http-server)
   npx http-server
   ```
   然後在瀏覽器中訪問 `http://localhost:8000`

### 使用步驟

#### 第一步：新增單字
1. 點擊上方導航欄的「**管理**」標籤
2. 在表單中輸入英文單字（例如：`hello`）
3. 點擊「**🤖 自動填入**」按鈕
   - 系統會自動查詢Dictionary API
   - 自動填入翻譯、詞性、例句等信息
4. 確認信息無誤後，點擊「**✓ 新增單字**」按鈕

#### 第二步：手動填入（可選）
- 如果想手動填入或修改信息：
  1. 在表單中填入所有欄位
  2. 直接點擊「**✓ 新增單字**」按鈕（跳過自動填入）

#### 第三步：開始學習
1. 點擊上方導航欄的「**學習**」標籤
2. **點擊卡片**可翻頁查看單字的翻譯和其他信息
3. 使用「**上一個**」和「**下一個**」按鈕瀏覽所有單字

#### 第四步：編輯或刪除
1. 回到「**管理**」頁面
2. 在單字列表中找到目標單字
3. 點擊「**✏️**」按鈕編輯，或點擊「**🗑️**」按鈕刪除

## 🔧 API 信息

本應用使用免費的 **Free Dictionary API**：
- **API 端點**：`https://api.dictionaryapi.dev/api/v2/entries/en/{word}`
- **優點**：
  - 無需驗證密鑰
  - 提供豐富的單字信息
  - 包括詞性、定義、例句、同義詞、詞源等
- **限制**：無

## 📋 表單欄位說明

| 欄位 | 說明 | 必填 |
|------|------|------|
| 英文單字 | 要學習的英文單字 | ✓ |
| 翻譯 | 中文翻譯 | ✓ |
| 詞性 | 單字的詞性（如 n., v., adj.） | ✗ |
| 例句 | 該單字在句子中的使用例子 | ✗ |
| 字根分析 | 同義詞、字根分析或其他補充信息 | ✗ |

## 🔐 隱私說明

- 所有數據存儲在本地瀏覽器中
- 不會將數據上傳到任何服務器
- 清除瀏覽器緩存時，本地存儲的數據會被刪除

## 🐛 故障排除

### 自動填入無法工作
- 檢查網絡連接
- 確認輸入的是有效的英文單字
- 某些單字可能在Dictionary API中不存在

### 數據丟失
- 檢查是否清除了瀏覽器緩存
- 某些瀏覽器的隱私模式可能不保存 localStorage

### API 錯誤
- 如果看到「查詢失敗」提示，可以手動填入信息
- 或稍後重試
